Custom Dark Mode is ONOFF

Enable custom dark mode so you can override settings for neutrals, accents and text colours.

Look for these visual cues to identify theme releated parameters

Saturation
Disable custom dark mode to restore common settings for neutrals, accents and text colours. Please note, it will reset your dark mode settings back to default
Learn more

Theme colours are generated programatically, and by default, they are the same in the light and dark modes. Custom dark mode allows you to fine-tune light and dark theme colours independently

Neurals

  • Saturation defines the intensity of the tone
  • Distance controls the difference between adjacent shades

Accents

  • Saturation makes it more or less vibrant
  • Luminance controls brightness and overall contrast

Text Colours

  • Accent text colour saturation enables custom saturation level for accent/semantic text

This creates consistency and uniformity. When you enable custom dark mode, the parameters above can be overridden to create a more nuanced dark mode.
